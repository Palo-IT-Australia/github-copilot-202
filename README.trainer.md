# Trainer README

## Purpose
This guide is for workshop trainers to quickly orient and facilitate sessions.

## Language Tracks (Quick Overview)
- JavaScript: cart and checkout flow, with tests in [exercises/javascript/README.md](exercises/javascript/README.md).
- TypeScript: typed user service and validation logic in [exercises/typescript/README.md](exercises/typescript/README.md).
- Python: data processing and legacy API hardening in [exercises/python/README.md](exercises/python/README.md).

## Customization Files Used in This Workshop
- Instruction file: [.github/copilot-instructions.md](.github/copilot-instructions.md)
  - Provides workshop context and guidance for AI assistance.
- Prompt file: [.github/prompts/workshop-readme-reviewer.prompt.md](.github/prompts/workshop-readme-reviewer.prompt.md)
  - Reviews workshop READMEs using a selected persona and focus.

## Facilitation Flow (Suggested)
1. Ask participants to choose one track based on language comfort.
2. Run exercises in sequence: instructions -> prompts -> skills -> agents -> hooks.
3. Validate each step with a runnable check (tests or command output).
